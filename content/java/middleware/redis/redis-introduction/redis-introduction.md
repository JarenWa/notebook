- ACID

    底层实现满足事务的原子性（Atomicity），一致性（Consistency），隔离性（Isolation），持久性（Durability）。

    K-V, Document, Graph......

- 无关联的
  
    通过 JSON 文档记录关联，有重复性。

- 非 SQL

    Redis, MongoDB, elasticsearch 语法都不一样。

- BASE
  
    无法全部满足 ACID。
