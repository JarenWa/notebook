# 1 概述 
上一节中, 我们只是输出一个简单的 HTML 页面, 就不得不编写上千行底层代码, 根本无法做到高效而可靠地开发

在 JavaEE 平台上, 由 Web 服务器完成处理 TCP 连接, 解析 HTTP 等底层工作. JavaEE 提供了 Servlet API, 程序员使用 Servlet API 编写自己的 Servlet 来处理 HTTP 请求, Web 服务器实现  Servlet API
